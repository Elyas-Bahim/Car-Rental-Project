# 🚗 Car Rent

A modern and responsive car rental website built using **HTML**, **CSS**, and **JavaScript**.

## 📖 About

Car Rent is a front-end web project that provides a modern and user-friendly interface for a car rental service. The website focuses on clean design, responsive layouts, and an intuitive user experience across different screen sizes.

## ✨ Features

* Responsive design
* Modern and clean user interface
* Interactive navigation
* Car listing section
* About section
* Services section
* Contact section
* Smooth scrolling experience
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/Elyas-Bahim/Car-Rental-Project.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

## 🎯 Purpose

The purpose of this project is to practice front-end web development by creating a responsive and visually appealing car rental website using core web technologies.

## 📱 Responsive Design

This website is fully responsive and optimized for:

* Desktop
* Tablet
* Mobile
